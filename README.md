# SFML | Sokoban
<g++|7.5.0>
<SFML|2.5.1>
<GNU Make|4.1>
Sokoban é um video-game de quebra-cabeça onde o jogador tem que organizar as caixas em seu devido local.
### Requisitos
 - Linux
 - G++
 - SFML
 - Terminal
### Guia de instalação 
## SFML
Em seu terminal linux execute:
```
sudo apt-get install libsfml-dev
```
## Compilar
Em seu terminal linux execute:
```
make exe
```
